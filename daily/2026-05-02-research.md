### 1. steipete/claude-code-mcp

**Type**: MCP server (nested agent bridge)

**What it actually does** (từ README):

- Output: Direct Claude Code execution with all permissions bypassed, multi-command queuing để tiết kiệm context
- Workflow: Agent calls `/mcp-claude:execute "refactor to async"` → MCP translates to Claude Code → returns results → queues multiple commands in single request
- **Real example**: Release automation: "Update version, create CHANGELOG, push tag" → queued together = 1 request instead of 3 → saves context cost

**Real example from README**:
```bash
# Refactoring function to async
/mcp-claude:execute "Convert function to async"

# Multi-step release (queued = lower context)

```


**Problem it solves**: Nested agents need file editing + system ops but can't directly access Claude Code → result is fragmented workflows. MCP wrapper = agents refactor code, manage files, version control, automate releases in one tool. Queued commands = conserve context vs sequential requests.

### Official: Claude Code Changelog — v2.1.126 (May 1-2, 2026)

**Source**: https://github.com/anthropics/claude-code/releases

**Type**: Official Anthropic release notes

**What changed** (key highlights):

**Model Picker Gateway Support** — `/model` picker now lists models from your gateway's `/v1/models` endpoint when `ANTHROPIC_BASE_URL` points to Anthropic-compatible server → supports custom model servers, local deployments

**Project Purge Command** — `claude project purge [path]` deletes all Claude Code state (transcripts, tasks, file history, config) with options:
- `--dry-run` (preview what deletes)
- `-y/--yes` (skip confirmation)
- `-i/--interactive` (selective deletion)
- `--all` (all projects)

**Enhanced Permission Bypass** — `--dangerously-skip-permissions` now bypasses prompts for:
- `.claude/` writes
- `.git/`, `.vscode/` changes
- Shell config edits
- Catastrophic rm still prompts for safety

**OAuth Improvement** — `claude auth login` accepts pasted OAuth codes when browser callback can't reach localhost → fixes WSL2, SSH, container auth issues

**Stability Fixes** — 30+ bug fixes:
- Security: Fixed `allowManagedDomainsOnly` ignored in config
- Images: Auto-downscale >2000px; remove oversized from history
- Windows: PowerShell 7 priority; fixed CJK text garbling
- Terminal: Fixed stream idle after sleep; long thinking pauses

**Real example from changelog**:
```bash
# Preview project cleanup
claude project purge ~/old-project --dry-run
# Output: Will delete 5 old transcripts, 23 tasks, stale config

# Gateway model support
export ANTHROPIC_BASE_URL="https://custom-gateway.com/v1"
/model
# Shows: Opus, Sonnet, Custom-FineTune-v1 (from gateway)

# Safe permission bypass for .claude/
claude --dangerously-skip-permissions -c "cp hook.sh ~/.claude/hooks/"
```

**Problem it solves**: Old projects accumulate session clutter → slow startup. Purge command = cleanup without manual deletion. Gateway support = custom servers visible in model picker. OAuth fix = WSL2/SSH/container users can now authenticate easily.
