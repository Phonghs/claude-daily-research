## 3. Awesome lists & Changelog updates

### Official: Claude Code Changelog — v2.1.121 (April 28, 2026)

**Source**: https://code.claude.com/docs/en/changelog

**Type**: Official Anthropic release notes

**What changed** (key highlights):

**`alwaysLoad` MCP Server Option** — When set to `true`, all tools from that MCP server skip tool-search deferral and are always available (vs lazy-loading)

**`claude plugin prune` Command** — Removes orphaned auto-installed plugin dependencies. `plugin uninstall --prune` now cascades to clean up related packages (prevents plugin bloat)

**Enhanced `/skills` Command** — Added type-to-filter search box for easier navigation in long skill lists (especially important with 2,849+ available skills)

**Improved Fullscreen Mode** — Typing into prompt no longer jumps scroll back to bottom after scrolling up. Dialogs that overflow terminal are now scrollable (arrow keys, PgUp/PgDn, Home/End, mouse wheel). Long URLs spanning multiple lines now fully clickable

**Terminal Clipboard Access (macOS)** — `/terminal-setup` now enables iTerm2's "Applications in terminal may access clipboard" setting. `/copy` command now works including from tmux

**MCP Server Improvements** — Transient errors during startup now auto-retry up to 3 times instead of staying disconnected (better reliability for slow/flaky servers)

**Notable Fixes** — Unbounded memory growth fixed when processing images. `/usage` memory leak fixed (~2GB potential recovery). Removed stale Recent Activity panel for faster startup. Improved LSP diagnostic summaries (expandable on click/Ctrl+O)

**Real example from changelog**:
```bash
# Plugin pruning
claude plugin uninstall my-old-plugin --prune
# Auto-removes dependencies no longer needed

# alwaysLoad in ~/.claude/settings.json
"mcp": {
  "myServer": {
    "alwaysLoad": true
    # All tools from myServer always available
  }
}
```

**Problem it solves**: Plugin dependencies accumulate → orphaned files waste disk/memory. `/skills` command buried in long lists. MCP servers timeout on startup → users restart. v2.1.121 fixes all three: auto-cleanup, search, auto-retry.
